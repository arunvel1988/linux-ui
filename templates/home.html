🐧 Linux Server Playground

Welcome to your Linux Server learning lab! This local playground helps you quickly create and manage Linux servers and desktops with one click:

  • 🛠️ Practice installing services using yum and apt commands
  • 🧰 Configure SSH, firewall (ufw/firewalld), users, groups, and permissions
  • 🗂️ Manage Nginx/Apache, FTP, Samba, and system logs
  • 💡 Automate tasks using cron, bash scripting, and systemd services
